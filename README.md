# gsky-tools

## Background

Common tools I use regularly

## Use

For debian-based:

`docker run --rm -it graboskyc/gsky-tools command` 

or for alpine-based:

`docker run --rm -it graboskyc/gsky-tools:slim command` 

## Valid Commands
* Generic
  * `busybox`
* MongoDB
  * `mongosh`
  * `mongocli`
  * `mongodump`
  * `mongoexport`
  * `mongofiles`
  * `mongoimport`
  * `mongorestore`
  * `mongostat`
  * `mongotop`
